# CI/CD com CodePipeline

[Continuous integration](https://aws.amazon.com/devops/continuous-integration/) (CI) e [continuous delivery](https://aws.amazon.com/devops/continuous-delivery/) (CD)
são essenciais para organizações hábeis. As equipes são mais produtivas quando podem fazer alterações discretas com frequência, liberar essas alterações programaticamente e entregar atualizações sem interrupções.

Neste módulo, construiremos um pipeline de CI/CD usando [AWS CodePipeline](https://aws.amazon.com/codepipeline/). O pipeline de CI/CD implantará um serviço de exemplo do Kubernetes, faremos uma alteração no repositório do GitHub e observaremos a entrega automatizada dessa mudança no cluster.
